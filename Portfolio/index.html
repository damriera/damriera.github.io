Boosted Botlaner

Boosted Botlaner is my own personal Discord Bot.

It is coded in Python using the discord.py library.

He is only in the Vladi's Supporters Incorporated Discord server (friends server with KC Vladi in it).

It has various features such as :

  • Songs Playback
  • Songs download from Spotify
  • Video and GIFs download from Twitter
  • Vladi's Spotify activity tracking and logging
  • Automated Twitch messages to inform viewers of the current song playing
  • Keywords detection and response
  • Commands the world must not be aware of

And I am still developing it to add more features and enhance the quality and performance of the bot.
